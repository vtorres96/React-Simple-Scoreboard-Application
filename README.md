 # Simple application developed using:
 ```sh  
 - class components 
 - props 
 - setState
 ```

## How to install packages (frontend)

### after cloning the repository enter the frontend folder

```sh
$ cd app
```

### yarn

```sh
$ yarn
$ yarn dev
```

### npm

```sh
$ npm install
$ npm start
```

## How to run application

### yarn

```sh
$ yarn dev
```

### npm

```sh
$ npm start
```

### Listening port on 3000

```sh
$ http://localhost:3000/
```
